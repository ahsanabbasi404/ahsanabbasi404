I build two things: **AI agents that do the work humans would rather skip**, and **the systems around them that don't fall over when real traffic shows up**. One half is deep, autonomous reasoning. The other half is making sure what I ship can actually be *found* — on Google, and inside the AI answers everyone now trusts more than Google.

Not chatbots. **Agents** — the kind that plan, call tools, recover when something breaks, and run multi-step workflows without a human babysitting every move.

My current stack runs on **Next.js, TypeScript, Node.js, and Python (FastAPI)**, with **OpenAI + LangChain** doing the thinking and **PostgreSQL / Redis / Docker** keeping it standing. Most days I'm somewhere between an agent's reasoning loop and a Lighthouse score that won't sit still.

For anything around **agentic systems, SaaS architecture, scalable backends, or SEO/GEO**, my inbox is open.
